# health_did_replication

NBER / working-paper version of Victor Rangel's Master's dissertation
"Do Health Policy Conclusions Survive Modern DiD? Re-estimating TWFE Studies"
(Insper, 2026, supervised by Cristine Pinto and Pedro H. C. Sant'Anna).

## Structure

```
health_did_replication/
├── main.tex          single-file main text (article class; chapters 1–5 inlined as sections)
├── AppendixA.tex     specification protocol and decision log
├── AppendixB.tex     per-article cards (event studies + HonestDiD)
├── AppendixC.tex     AI-assisted pipeline architecture
├── AppendixD.tex     article collection pipeline
├── example.bib       BibLaTeX bibliography
├── Figures/          all figures referenced by main.tex and appendices
└── README.md
```

No custom document class, no `Chapters/` subfolder, no `Appendices/` subfolder — a
plain `article` document plus sibling appendices.

## Build

```
pdflatex main
biber main        # or bibtex main
pdflatex main
pdflatex main
```

Requires the standard LaTeX packages listed in the preamble of `main.tex`
(`amsmath`, `graphicx`, `booktabs`, `threeparttable`, `tcolorbox`, `biblatex`,
`tikz`, `subcaption`, etc.). A modern TeX Live distribution has all of them.

## Relationship to the dissertation

Content is identical to the dissertation version shipped in `overleaf/` in the
same replication package. The differences are purely in format:

| Dimension | Dissertation (`overleaf/`) | Working paper (this folder) |
|-----------|----------------------------|-----------------------------|
| Document class | `MastersDoctoralThesis_dry_en` (custom) | `article` (standard) |
| Top-level sectioning | `\chapter{}` | `\section{}` |
| Chapter files | 5 separate `Chapters/Chapter*.tex` | inlined in `main.tex` |
| Appendix files | 4 files in `Appendices/` | 4 files at root, named `Appendix*.tex` |
| Title page | Insper dissertation template | Plain article `\maketitle` |
| Acknowledgements | Dedicated page | First-page `\thanks{}` footnote |
| TOC / List of figures / List of tables | Present | Omitted |
| Abstract | University-style environment | Article-class environment |
| Authors | Single-author | `Victor Rangel · (co-author TBA)` |
| Keywords / JEL codes | Not shown | Shown below the abstract |

## Updating this version when the dissertation is revised

When the dissertation is updated, sync from the canonical `overleaf/` project
manually:

1. For each chapter file `overleaf/Chapters/Chapter{1..5}.tex`, demote the
   headings one level (`\chapter` → `\section`, `\section` → `\subsection`,
   etc.) and concatenate the result into the body of `main.tex` (between the
   abstract and the references).
2. For each appendix file `overleaf/Appendices/Appendix{A..D}.tex`, demote the
   headings in the same way and write the result to `AppendixX.tex` at the
   root of this folder (overwriting the previous version).
3. Copy `overleaf/Figures/` on top of the local `Figures/` (new figures, same
   filenames).
4. Copy `overleaf/example.bib` to the root if bibliography entries were added.

Do not touch the preamble of `main.tex` during a sync — that part is specific
to the working paper format.

All figures and tables referenced here are regenerated by the code and data in
the replication package root. To regenerate them from scratch:

```
cd "Replication Package"

```
